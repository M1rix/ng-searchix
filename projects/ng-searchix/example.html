ngx-searchix New Feature

Recent Items with localStorage Support

🎉 New Feature: Recent Items

  • Shows recent items when search input is empty
  • Click × button to remove items from recents
  • Selected items automatically added to recents
  • Automatically saved to localStorage
  • Maximum 10 recent items
  • Works with both provided recentItems and localStorage

📋 Demo 1: Provided Recent Items

Passing recentItems as input. Changes are saved to localStorage.

✨ How It Works

  • Empty Search: Shows recent items instead of all items
  • On Select: Item automatically added to recents (top of list)
  • On Remove: Click × button to remove from recents
  • No Duplicates: Selecting same item moves it to top
  • Automatic Limit: Only keeps last 10 recent items
  • Persistent: Saved to localStorage automatically
  • Flexible: Works with or without provided recentItems

🎨 UI Features

  • "Recent" header with clock icon when showing recents
  • Delete button (×) appears on hover over recent items
  • Smooth animations for button appearance
  • Red hover effect on delete button
  • External link icon still works alongside delete button

💻 Complete Usage Example

// app.component.ts import { Component } from '@angular/core'; import { SearchItem } from 'ngx-searchix'; @Component({ selector: 'app-root', templateUrl: './app.component.html' }) export class AppComponent { // Your search items items: SearchItem[] = [ { id: '1', title: 'Getting Started', subtitle: 'Learn the basics', icon: 'rocket' }, { id: '2', title: 'API Reference', subtitle: 'Complete documentation', icon: 'book' } // ... more items ]; // Optional: Provide initial recent items // If omitted, will load from localStorage automatically recentItems: SearchItem[] = [ { id: '1', title: 'Getting Started', subtitle: 'Learn the basics', icon: 'rocket' } ]; onItemSelected(item: SearchItem) { console.log('Selected:', item); // Item is automatically added to recents! } } // app.component.html <ngx-searchix [items]="items" [recentItems]="recentItems" placeholder="Search documentation..." hotkey="ctrl+k" (itemSelected)="onItemSelected($event)" ></ngx-searchix> // Or without recentItems (auto localStorage): <ngx-searchix [items]="items" placeholder="Search..." ></ngx-searchix>

🔑 localStorage Key

  • Key: searchix-recents
  • Defined in: SEARCHIX_RECENTS_KEY token
  • Format: JSON array of SearchItem objects
  • Auto-managed: No manual intervention needed
⚡ Pro Tip:

You don't need to manage recents manually! Just use the component and it will automatically:

1. Add selected items to recents

2. Remove items when × is clicked

3. Save everything to localStorage

4. Load from localStorage on next visit

📖 This is a documentation example. To see the actual working demo, build and run the Angular application.
